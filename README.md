<h1>Valorant Bot plugin</h1>

<b>v0.6.0</b>

Miria Console 插件
需要安装 Sqlite3

1.支持查询每日商店, 配件商店.

2.每日商店定时推送.

3.支持skiko或java-awt绘图.

关于系统是否支持skiko,可以用脚本[OsTest.py](OsTest.py)跑一下.如果正常输出了系统架构就代表skiko可以运行.

更新内容记录在[Changelog.txt](CHANGELOG.txt)Changelog.txt中,或者查看Release记录.
